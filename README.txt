# For Flock's Sake
### ...being a game of collection, and also birds.

Control using the arrow keys (or, if you have it, a joystick/gamepad). 
Navigate around the world, collecting birds into your flock and returning them to
the safety of the fountain in the center.

## HELP

Requires Python 2.6, PyOpenGL 3.x, and PyGame 1.9.x. Sorry.

Created for [Global Game Jam 2011](http://archive.globalgamejam.org/2011/flocks-sake) by a team of people.
